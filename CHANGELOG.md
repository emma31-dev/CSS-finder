# Change Log

All notable changes to the "css-finder" extension will be documented in this file.

## [0.0.1] - 2024-11-23

### Added
- Initial release
- CSS class scanning functionality
- Support for HTML, JS, JSX, TS, TSX, and Vue files
- Configurable ignore folders
- Configurable file extensions
- Output panel for results
- Progress notifications during scanning
